# ⚡ Latency Killer & RAM Purger
A professional system optimization tool designed for competitive gaming and low-latency environments.

## 🚀 Key Features
- **0.5ms System Timer Lock:** Drastically reduces input lag by forcing Windows to its minimum timer resolution.
- **Intelligent Standby List Purge:** Automatically clears cached memory based on user-defined limits to prevent micro-stutters.
- **Multi-Game Process Optimization:** Elevates game process priority to High and locks CPU Affinity to all available cores (optimizing P-Core usage).
- **Silent Background Operation:** Can be configured to launch minimized to the system tray on startup.

## ⚙️ How to Use
1. Run the application as **Administrator**.
2. Add your game executables (e.g., `cs2.exe`) via the **ADD** button.
3. Set your RAM limits (e.g., 1024MB) for automatic purging.
4. Check **GAME MODE** to enable real-time process optimization.
